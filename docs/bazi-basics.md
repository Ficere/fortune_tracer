# 命理学基础

## 四柱八字

四柱即年柱、月柱、日柱、时柱，每柱由一个天干和一个地支组成，共八个字。

- **天干**：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- **地支**：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

## 五行

### 相生

木 → 火 → 土 → 金 → 水 → 木

### 相克

木 → 土 → 水 → 火 → 金 → 木

## 十神

基于日主（日柱天干）与其他天干的五行生克关系：

| 关系 | 同性 | 异性 |
|------|------|------|
| 同我 | 比肩 | 劫财 |
| 生我 | 偏印 | 正印 |
| 我生 | 食神 | 伤官 |
| 克我 | 七杀 | 正官 |
| 我克 | 偏财 | 正财 |

## 辅助宫位

- **命宫**：先天禀赋和性格特质
- **胎元**：受胎时的能量场
- **身宫**：后天发展和行为模式

## 神煞

- **吉神**：天乙贵人、文昌、将星、禄神等
- **凶煞**：羊刃等

## 大运

每十年一步大运，顺逆取决于阴阳年和性别。
